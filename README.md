This repository is created for the **Software Construction and Development (SCAD)** course of 4rth semester to demonstrate the use of **Git and GitHub** for version control and collaboration.  
It includes basic Git commands, branch management, pull requests, and merge operations.

Learning Objectives
- Understand how Git tracks changes in a project.  
- Practice creating and switching between branches.  
- Learn how to create pull requests and merge branches.  
- Collaborate effectively using GitHub.

Repository Features
- Demonstrates common Git workflows (`add`, `commit`, `merge`, `push`, `pull`).  
- Shows how to handle multiple branches and merge them into `main` or `master`.  
- Includes a sample pull request reviewed and merged for collaboration practice.
